# CodeGuard AI - Vercel Environment Setup

## 1. No Painel do Vercel

Acesse https://vercel.com/dashboard e selecione seu projeto.

### Environment Variables

Configure as seguintes variáveis no painel do Vercel:

```
CODEGUARD_API_KEYS=prod-key-1,prod-key-2,prod-key-3
CODEGUARD_LICENSE_KEY=your-enterprise-license-key
OPENAI_API_KEY=sk-proj-...
ANTHROPIC_API_KEY=sk-ant-api03-...
SILICONFLOW_API_KEY=your-siliconflow-key
KIMI_API_KEY=your-kimi-key
STRIPE_WEBHOOK_SECRET=whsec_...
```

## 2. Via Vercel CLI

```bash
# Login
vercel login

# Adicionar variáveis
vercel env add CODEGUARD_API_KEYS
# Digite: prod-key-1,prod-key-2,prod-key-3

vercel env add CODEGUARD_LICENSE_KEY
# Digite sua license key

vercel env add OPENAI_API_KEY
# Digite sua OpenAI key
```

## 3. Verificação

Após configurar, teste com:

```bash
curl -X POST https://your-app.vercel.app/api/scan \
  -H "x-api-key: prod-key-1" \
  -H "Content-Type: application/json" \
  -d '{"region": "BR"}'
```

## 4. Segurança

### API Keys
- Use chaves diferentes para produção e desenvolvimento
- Rotacione chaves a cada 90 dias
- Monitore uso no Vercel Analytics

### Rate Limiting
- Implementado automaticamente (100 req/15min)
- Monitore headers de rate limit

### Logs
- Acesse logs em tempo real: `vercel logs`
- Configure alertas para códigos 5xx

## 5. URLs de Produção

Após deploy, suas APIs estarão em:
- `https://your-app.vercel.app/api/scan`
- `https://your-app.vercel.app/api/graph`
- `https://your-app.vercel.app/api/shadow-apis`
- `https://your-app.vercel.app/api/openapi`
- `https://your-app.vercel.app/api/docs`

## 6. Custom Domain (Opcional)

Para usar domínio próprio:

```bash
vercel domains add your-domain.com
# Configure DNS no seu provedor
```

## 7. Preview Deployments

Cada PR cria um preview deployment:
- URLs: `https://pr-123-your-app.vercel.app`
- Teste APIs em staging antes de produção
- Configure branch protection rules
